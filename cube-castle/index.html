Cube Castle

Hello, and welcome to an amazing addition to the original Cube Castle which was written in Python. (You can find that here) Because I wanted to go and remake it in Javascript. That is what this is. Enjoy! (The font used in the score is called Tokyo Midnight. By The Letters Guy). Now introducing Cube Castle: The App! Make sure you have at least Java 21 installed. Also make sure to put this jar file in the same directory before running the program. You can easily run it by double-clicking once you have java installed.
